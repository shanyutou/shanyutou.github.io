Java lambda表达式-箭头函数

Aug 2, 2020
java

jdk8+以上引入了箭头函数，即J8的lambda表达式。

Lambda表达式的本质只是一个”语法糖“，由编译器推断并帮你转换包装为常规的代码，因此你可以使用更少的代码来实现同样的功能。

本人建议不要乱用,因为这就和某些很高级的黑客写的代码一样,简洁,难懂,难以调试,维护人员想骂娘，哈哈
